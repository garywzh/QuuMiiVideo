# QuumiiBox

趣觅盒子是[火鸡岛](http://www.huoji.tv)的一个第三方客户端，数据来源于火鸡岛。

下载应用请戳

[![Get it on Google Play](google-play-badge.png)](https://play.google.com/store/apps/details?id=org.garywzh.quumiibox)

or

 http://fir.im/16ml

##FEATURE

支持以下功能：

*登录

*点赞、点踩

*收藏

*评论

*分享

*视频播放竖屏/全屏切换

*添加自定义标签快捷方式

*关键词搜索
