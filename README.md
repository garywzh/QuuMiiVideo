# QuumiiBox

趣觅盒子是[趣觅](http://www.quumii.com)的一个第三方客户端，数据来源于趣觅网。

下载应用最新版本请戳 http://fir.im/16ml

##FEATURE

*由于展示效果的原因，来自部分视频源（如秒拍）的视频会转到浏览器播放。
